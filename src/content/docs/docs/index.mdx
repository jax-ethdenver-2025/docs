---
title: jax 
description: Learn about the jax protocol - the permissionlessly incentivized file storage layer for Ethereum
---

jax aims to provide a decentralized storage network where anyone can participate 
as a storage provider and earn rewards. Built on Ethereum + EigenLayer
it ensures your files remain accessible, censorship-resistant, and secure.

## Goal Features

<ul class="space-y-2 mt-4">
  <li class="flex items-start">
    <span class="inline-block mr-2 text-white">•</span>
    <span><strong class="text-white">Decentralized Storage:</strong> Files are chunked and distributed across multiple peers</span>
  </li>
  <li class="flex items-start">
    <span class="inline-block mr-2 text-white">•</span>
    <span><strong class="text-white">Ethereum Native:</strong> Storage contracts execute on Ethereum</span>
  </li>
  <li class="flex items-start">
    <span class="inline-block mr-2 text-white">•</span>
    <span><strong class="text-white">Decoupled Incentives:</strong> Consensus is decoupled from the rewards model. Contracts can describe any incentives structure they want.</span>
  </li>
  <li class="flex items-start">
    <span class="inline-block mr-2 text-white">•</span>
    <span><strong class="text-white">Verifiable Availability:</strong> Jax nodes arrive at consensus over the state of the network and can agree on how to reward storage providers.</span>
  </li>
</ul>
